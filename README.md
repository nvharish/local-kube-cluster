# kubeadm-clusters

This repository contains scripts to create Kubernetes clusters on various platforms.

## Overview

Use the scripts in this repository to quickly set up and deploy Kubernetes clusters across different infrastructure providers and environments.

## Features

- Multi-platform cluster creation
- Automated deployment scripts
- kubeadm-based cluster initialization

## Getting Started

1. Clone this repository
2. Review the scripts for your target platform
3. Follow the platform-specific instructions

## Platforms Supported

- [Windows](https://github.com/nvharish/kubeadm-clusters/windows)

## Directory Structure

```
kubeadm-clusters/
├── windows/           # Windows-specific scripts and configurations
├── aws/               # AWS-specific scripts and configurations
├── linux/             # Linux-specific scripts and configurations
└── README.md          # Instructions on platform specific usage.
```
